# Creating-a-Custom-ISO-with-Preinstalled-Applications
<h1>Steps to build the custom iso</h1><br><br>
<p>1. Choose a base operating system(Debian, NixOS or Fedora).</p><br>
<p>Here Fedora is used.</p><br><br>
<p>2. Setup Environment:</p><br>
<p>--install fedora live cd with standard kickstart file</p><br>
<p>---sudo dnf install livecd-tools spin-kickstarts curl wget</p><br>
<h1>Installation</h1><br><br>
<p>1.Install any vm software. Here VMware is used to boot. link:https://www.vmware.com/products/desktop-hypervisor.html<br>
<p>2.Install Fedora. link:https://fedoraproject.org/workstation/download<br><br>
<h1>Verification steps to check the functionality of preinstalled applications.</h1><br><br>
<p>booting the custom iso file in a virtual machine or physical hardware.<br>
<p>check if the application is working properly or not by using 'super' key.
